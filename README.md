# LibraNet

## Proje Hakkında

![LibraNet](C127D2B5-FFF1-4F14-A8CB-228306B7EE04.png)

### Proje Tanımı

LibraNet, kütüphane yönetim süreçlerini dijital ortama taşımak için tasarlanmış bir REST API sistemidir. Kullanıcılar sisteme kayıt olarak giriş yapabilir, kütüphanedeki kitapları görüntüleyebilir ve rezervasyon oluşturabilir. Admin rolündeki kullanıcılar ise sisteme kitap ekleyebilir, güncelleyebilir ve silebilir. LibraNet; sade mimarisi, rol tabanlı yetkilendirme sistemi ve rezervasyon yönetimiyle kütüphane işlemlerini hızlı şekilde yönetmeyi hedeflemektedir.
